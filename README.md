## How to install

1. Clone this repository:

```bash
git clone https://github.com/dipyamanroy/enigma-lander.git
```

2. Go into project

```bash
cd enigma-lander
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
